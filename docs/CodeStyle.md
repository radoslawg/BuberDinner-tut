
# Conventions

- `usings` section should be inside namespaces
- System usings should come first
- Use primary constructors if possible

## Namespaces

Prefer not scoped namespaces i.e.

```csharp
namespace Somthing.Somethingelse;
```

instead of:

```csharp
namespace Somthing.Somethingelse 
{
...
}
```
